## 📝 版本号说明

LightSearch 采用语义化版本号系统，格式为：`vX.Y.Z_SUFFIX`

- **X.Y.Z**: 主版本号.次版本号.修订号
- **SUFFIX**: 版本后缀
  - `_A + 数字`: 开发版/测试版（如 `_A0001` 表示第一个测试版）
  - `_B`: 改进版（Bug修复和小改进）
  - `_C`: 候选发布版
  - 无后缀: 正式稳定版

### 示例
- `v2.1.0` - 正式版本
- `v2.1.0_A0001` - 第一个开发测试版
- `v2.1.0_A0002` - 第二个开发测试版
- `v2.1.0_B` - 改进版（基于v2.1.0的改进）
- `v2.1.1` - 下一个正式版本

#### 4. 标题修复
- ✅ 修复标题自动变回"title"的异常
- ✅ 使用 MutationObserver 监听标题变化
- ✅ 定期检查并修正标题
- ✅ 确保标题始终为正确值

**技术方案**:
```javascript
// 监听标题元素变化
const titleObserver = new MutationObserver((mutations) => {
    if (document.title !== correctTitle) {
        document.title = correctTitle;
    }
});

// 定期检查（备用方案）
setInterval(() => {
    if (document.title !== correctTitle) {
        document.title = correctTitle;
    }
}, 1000);
```
