## 📦 本次更新

### 🆕 新功能

#### 1. 首次访问引导系统
- 智能检测用户首次访问
- 自动在新标签页打开介绍站
- 主站显示欢迎弹窗，引导用户查看使用手册
- 支持多语言欢迎消息

#### 2. 设置面板增强
新增5个快捷入口：
- 🏠 **跳转到官方页** - 快速访问介绍站
- 📖 **用户使用手册** - 查看完整文档
- 💻 **GitHub项目页** - 访问源代码仓库
- 🐛 **从GitHub提交问题** - 报告Bug或建议
- ℹ️ **关于** - 查看版本信息和开发者信息

#### 3. 关于对话框
- 显示当前版本号
- 开发者主页链接
- 开源协议信息
- 一键清除缓存功能
- 调试模式入口（预留）

#### 4. 主题切换功能
- **介绍页**: 右上角添加明亮/黑暗模式切换按钮
- **文档编辑器**: 顶部工具栏主题切换
- 主题偏好自动保存
- 平滑过渡动画

### 🐛 Bug修复

#### 标题异常修复
- 修复页面标题自动变回"title"的问题
- 使用MutationObserver监听标题变化
- 添加定期检查机制确保标题正确
- 在控制台输出警告信息便于调试

### 🔧 改进优化

- 优化欢迎弹窗样式和动画
- 改进主题切换的视觉效果
- 完善多语言支持（英文、中文、日文）
- 优化代码结构和可维护性
- 添加详细的代码注释

## 🚀 升级指南

### 从 v2.1.0 升级

#### 方法一：完整替换（推荐）
1. 备份当前设置（可选）
   ```javascript
   // 在浏览器控制台执行
   const backup = JSON.stringify(localStorage);
   console.log(backup); // 复制保存
   ```

2. 下载最新版本
   - [GitHub Release](https://github.com/StarsailsClover/LightSearch/releases/tag/v2.1.0_B)
   - [直接下载ZIP](https://github.com/StarsailsClover/LightSearch/archive/refs/tags/v2.1.0_B.zip)

3. 替换文件
   - 解压下载的文件
   - 替换整个项目目录

4. 清除浏览器缓存
   - Chrome/Edge: `Ctrl + Shift + Delete`
   - Firefox: `Ctrl + Shift + Delete`

5. 刷新页面
   - 按 `F5` 或 `Ctrl + R`

#### 方法二：手动更新
只更新必要文件：

1. 下载以下文件：
   - `script-v2.1.0_B.js`
   - `Introducing/dark-mode.js`

2. 更新 `index.html`：
   ```html
   <!-- 在 </body> 前添加 -->
   <script src="script-v2.1.0_B.js"></script>
   ```

3. 更新 `Introducing/index.html`：
   ```html
   <!-- 在 </body> 前添加 -->
   <script src="dark-mode.js"></script>
   ```

4. 清除缓存并刷新

## ⚠️ 注意事项

### 首次访问行为
- 升级后首次访问会被识别为"首次访问"
- 会自动打开介绍站并显示欢迎弹窗
- 这是正常行为，不会影响后续使用

### 缓存清除
- "清除缓存"功能会重置所有设置
- 包括搜索引擎配置、主题偏好等
- 执行前请确认是否需要备份

### 主题偏好
- 主题选择会自动保存到localStorage
- 不同页面的主题设置是独立的
- 介绍页和主站的主题可以不同
