## 📝 使用说明

### 启用 LiquidGlass 模式
1. 打开"设置"
2. 在"主题"下拉框选择"Liquid Glass"
3. 调整参数（可选）
4. 保存设置

### 使用显示模式自动切换
1. 打开"设置"
2. 在"显示模式"选择"自动（按时间）"
3. 设置切换时间：
   - 明亮→黑暗：默认 18:00
   - 黑暗→明亮：默认 06:00
4. 保存设置

### 上传自定义 Logo
1. 打开"设置"
2. 点击"Custom Logo" 的 "Upload"
3. 选择图片
4. 如果比例不符，会自动打开裁剪器
5. 拖动裁剪区域到合适位置
6. 点击"保存"

### 查看错误日志
1. 打开"设置"
2. 滚动到底部
3. 点击"View Error Log"
4. 查看所有错误记录
5. 可以点击"Clear Log"清空
