## 📋 版本控制

### Git 提交建议

```bash
# 添加新文件
git add script-v2.1.0_B.js
git add Introducing/dark-mode.js
git add docs/posts/progress/

# 添加修改的文件
git add index.html

# 提交
git commit -m "Release v2.1.0_B: 首次访问引导、设置增强、主题切换"

# 打标签
git tag -a v2.1.0_B -m "LightSearch v2.1.0_B Release"

# 推送
git push origin main
git push origin v2.1.0_B
```

- `v2.1.0` - 上一个正式版本
- `v2.1.0_B` - 当前版本（改进版）
- `v2.1.1` - 下一个计划版本
