## 📝 版本号说明

LightSearch 采用语义化版本号：`vX.Y.Z_SUFFIX`

- **X.Y.Z**: 主版本.次版本.修订号
- **SUFFIX**: 版本后缀
  - `_A + 数字`: 开发版/测试版
  - `_B`: 改进版
  - `_C`: 候选发布版
  - 无后缀: 正式稳定版

### 示例
- `v2.1.0` - 正式版本
- `v2.1.0_A0001` - 第一个开发测试版
- `v2.1.0_B` - 改进版（当前版本）
- `v2.1.1` - 下一个正式版本
